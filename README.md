```
- Panorama Builds -
2023 - Works fine, supported
2022 - Untested, should work fine
2021 - Works fine, supported
2020 - Tested, Stable
2019 - Tested, Stable
Late 2018 - Tested, Stable
- Scaleform Builds -
2015-2018 - Very unstable for now, not recommended
```
